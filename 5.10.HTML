5.10 Random Numbers

Notes

Intro

JavaScript provides a number of functions that are useful for generating random numbers. Math.random() is a function that returns a pseudo-random number between 0 and 1, not including 1 Math.floor() is a function that takes a decimal number and rounds it down - chops off the decimal

To produce a random integer from min value to max value (including the min and max) use this calculation Math.floor(math.random() * (max + 1)) + min

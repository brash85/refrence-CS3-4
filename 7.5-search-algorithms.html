7.5 Search Algorithms

Objectives

To develop and write linear and binary search algorithms with guidance
To be able to describe the efficiency of an algorithm using "Big O" notation

Notes

Linear (or sequential) search checks each element in an array for a target vaule from start (i=0) to end (i = length - 1) until the target is found or the end of the array is reached. STEPS 1 set array and target value. 2 set a counter to i = 0 3 check if the element at arr[1] is equal to the target value 4
